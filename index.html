Intoxication Detection Using Speech

Intoxication Detection Using Speech

Github

"Intoxication Detection Using Speech" is focused on leveraging machine learning techniques, specifically Convolutional Neural Networks (CNN) and Mel-Frequency Cepstral Coefficients (MFCC), to detect intoxication by analyzing speech patterns. Published in IEEE, this project aims to provide a non-invasive and accurate approach to intoxication detection, eliminating the need for traditional methods like breathalyzers. By examining various aspects of speech signals and employing advanced feature extraction methods, the model achieves an impressive accuracy of 96.6% with MFCC and 96% with CNN.
